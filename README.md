# Modern Trials: Bespoke Landing Page

A single-file bespoke landing page prepared for Modern Trials, built in Modern Trials' own visual register (deep purple gradient, Poppins display, Open Sans body, compliance badge band echoing their homepage) so it reads as something their own team could have produced. The page carries Eduba's fit-analysis pitch body as a short read on where the throughput wall sits between the Trial Recruitment Toolbox and the next set of sponsor deals, frames the 60/30/10 orchestration split against their stack, surfaces the matching Feeld engagement and the Ethics Engine governance paper as proof, and closes on a 30 minute calendar link with Matt Creamer (CRO). Eduba branding is restricted to the footer and a single governance line about the NLP Logix partnership. Mobile-first responsive down to 375px, reduced-motion aware, and ships with no pricing, no hype, and no em dashes.
